Interactive DSA Roadmap

How to View & Run This Roadmap in Visual Studio Code

1. Save this HTML as dsa-roadmap.html:
- In VS Code, create a new file and paste the HTML code.
- Save it in a folder of your choice with the name dsa-roadmap.html.

2. Install and Open with Live Server:
- Open the folder in VS Code (File → Open Folder…).
- Go to Extensions (Ctrl+Shift+X or Cmd+Shift+X).
- Search for Live Server by Ritwick Dey and install it.
- In the Explorer, right-click dsa-roadmap.html → “Open with Live Server”.

3. See It Live:
- A browser tab will open automatically. Whenever you save changes in VS Code, the page refreshes.
- If you don’t have Live Server, just double-click dsa-roadmap.html in your file explorer to open it in the browser and refresh manually after edits.
